# Distribucion-de-probabilidad-y-prueba-de-hipotesis

Aplicación interactiva desarrollada con Streamlit para análisis
estadístico con asistente de IA integrado.

Proyecto para la materia de Probabilidad y Estadística.

## ¿Qué hace?
Permite cargar datos (CSV o sintéticos) y realizar:
- Visualización de distribuciones (histograma, KDE, boxplot)
- Pruebas de hipótesis Z con interpretación automática
- Análisis asistido por IA vía Google Gemini API

## Tecnologías
- Python
- Streamlit
- Google Gemini API
- Pandas / NumPy / Matplotlib / SciPy

## Cómo ejecutarlo
```bash
git clone https://github.com/sunsytt/[nombre-repo]
cd [nombre-repo]
pip install -r requirements.txt
streamlit run app.py
```

## Funcionalidades
- Carga de datos CSV o generación sintética
- Visualización de distribuciones con análisis de normalidad
- Prueba Z (bilateral, cola izquierda, cola derecha)
- Interpretación automática de resultados
- Asistente IA para análisis estadístico
